----------------------------
Database(MySQL)
---------------------------
10. Install xampp or any MySQL Database server of your choice

11. Import the JIMS.sql Database and set the username, password and hostname in db.js file.

----------------------------
Backend (Express)
----------------------------

12. In the 'backend' directory:

 ```
 npm start
 ```

 This will start the Express server.

13. Your backend API will be available at http://localhost:5000.
